# Kotlin Grade Tracker

Simple program for tracking students and their grades in different classes. Used as an exercise in learning Kotlin.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Simple command line program made in IntelliJ IDEA. Simply open the Main.kt file, compile and run it

Instructions for using the software:

1. After intalling, use file > Open and open the Main.kt file
2. Use the green play button to run the program

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Written in IntelliJ IDEA 2025.3.2 with the default libraries
* It's a simple program so it probably works with almost any version.
